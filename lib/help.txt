Usage: npx flecss <path:source> <path:target> [--<arg>|-<arg:shorthand> *?]*

Flags:
--standalone, -S        Build without including flecss.
--watch, -W             Watch file changes for incremental builds.

Options:
--library, -L           Specify the flecss library to compile with (see above, default is `flecss.css`)
