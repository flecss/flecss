Usage: npx flecss <path:source> <path:target> [--<arg>|-<arg:shorthand> *?]*

Flags:
--print-stacktrace      Print stack trace of errors.
--watch, -W             Watch file changes for incremental builds.

Options:
--library, -L           Specify the flecss library to compile with (default is 'flecss').
--config, -C            Specify path to a file that contains overrides for flecss variables  (.scss, or .json).
